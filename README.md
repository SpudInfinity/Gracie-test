# Gracie Abrams Song Quiz 🕯️

A minimalist, atmospheric song quiz inspired by Laufeyland, featuring the haunting sounds of *Block Me Out*.

## Features
- 🎵 Immersive background music (Syncs with start button)
- 🕊️ Aesthetic UI with glassmorphism effects
- 📱 Fully responsive for mobile devices
- 🎹 Curated lyrics and results based on your personality
